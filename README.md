# Car Damage Severity Predictor

Image classification model that predicts car damage severity (minor, moderate, severe) using EfficientNet-B0 with transfer learning.

## Model Details

- **Architecture:** EfficientNet-B0 (transfer learning from ImageNet)
- **Classes:** minor, moderate, severe
- **Training accuracy:** ~80%
- **Validation accuracy:** ~73%
- **Dataset:** [Car Damage Severity Dataset](https://www.kaggle.com/datasets/prajwalbhamere/car-damage-severity-dataset)

## Setup

```bash
pip install -r requirements.txt
```
